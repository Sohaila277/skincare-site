#  Skincare Landing Page

A premium, modern React landing page designed for a high end skincare and beauty brand. This project focuses on clean aesthetics, simple UI elements, and a seamless user experience.

---

##  Features

- **Slim Navigation Bar** – Smooth-scroll enabled links with professional React Icons.
- **Hero Section** – High-impact visual entry point with brand messaging.
- **Services & Treatments** – Categorized grid layouts showcasing brand offerings.
- **Thematic Map Integration** – Custom-styled, grayscale Google Map integration with a branded contact overlay.
- **Top Products Section** – Featured product showcase with price points and call-to-action buttons.
- **Professional Footer** – Multi-column coffee-brown footer featuring social integration and quick links.
- **Fully Responsive** – Mobile-first design that looks great on all devices.

---

## Tech Stack

- **React 18** – Frontend Framework  
- **Vite** – Build Tool  
- **React Icons** – Professional Iconography  
- **CSS3** – Custom styling with Flexbox, Grid, and Glassmorphism effects  
- **GH-Pages** – Deployment and hosting

##  Local Setup

If you'd like to run this project locally, follow these steps:

### 1️ Clone the repository

```bash
git clone https://github.com/sohaila277/skincare-site.git
```

### 2️ Navigate to the project folder

```bash
cd skincare-site
```

### 3️ Install dependencies

```bash
npm install
```

### 4️ Start the development server

```bash
npm run dev
```

The app should now be running at:

```bash
http://localhost:5173/
```

##  Map Integration Note

The map section uses a static image overlay with CSS filters (`grayscale` and `multiply`) to achieve a professional, branded look without the heavy load times of a live API.
